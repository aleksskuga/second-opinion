# Second Opinion — Второе мнение в родах

Мобильное веб-приложение для рожениц и их партнёров. Помогает критически оценить предложения врача во время родов и принять осознанное решение: соглашаться на вмешательство или можно подождать.

## Что это

Два инструмента, которые работают прямо в браузере телефона без установки и интернета:

1. **Справочник** (`akusher_prod_v5.html`) — интерактивный помощник для использования прямо в родах
2. **Тренажёр** (`trainer.html`) — симулятор для подготовки до родов: учит оценивать предложения врача и принимать осознанные решения

Не заменяет врача — даёт структуру для диалога с ним.

## Тренажёр принятия решений

Интерактивный симулятор, в котором вы:
- Получаете клиническую ситуацию — что говорит врач и что наблюдаете
- Решаете: **подождать**, **согласиться** или **задать вопросы по ПРАН**
- Получаете разбор: почему это верно или нет, и какие вопросы стоило задать

35 клинических сценариев с балансом: в ~половине случаев врач прав и нужно соглашаться, в ~половине — стоит задать вопросы и подождать. Это формирует правильный паттерн мышления: не соглашаться на всё подряд, но и не отказываться от нужной помощи.

Два режима: полный (35 сценариев) или быстрый (10 случайных).

## Справочник

### Метод ПРАН
Фреймворк из 4 вопросов врачу перед любым вмешательством (окситоцин, амниотомия, эпидуральная, кесарево):
- **П**ольза — зачем это именно мне сейчас?
- **Р**иски — какие риски у вмешательства?
- **А**льтернатива — есть ли другой способ?
- **Н**ичего — что будет, если подождать?

### 35 клинических сценариев в тренажёре

Сценарии сбалансированы — в ~половине врач прав (нужно соглашаться), в ~половине можно подождать:

| Тема | Подождать | Согласиться |
|------|-----------|-------------|
| Стимуляция / окситоцин | Латентная фаза, пересменка врачей | Истинная остановка активной фазы |
| Кесарево | Пограничная КТГ при 9 см | Стойкая брадикардия, патологическая КТГ |
| Зелёные воды | Слегка зеленоватые, плод стабилен | Тёмный меконий + дистресс |
| Амниотомия | «Для ускорения» при высокой головке | — |
| Безводный период | 14 ч, всё стабильно | Инфекция (38.5°C), 48 ч с ростом температуры |
| Затяжные роды | 4 см / 2 часа наблюдения | 6 см / 5 часов, всё перепробовано |
| Фаза замедления | Пауза при 8 см | — |
| Потуги | Тужиться рано при 10 см | 3 часа без продвижения |
| Эпидуральная | Без показаний | Дискоординация |
| Обвитие / пуповина | Петли по УЗИ, норм. длина | Выпадение пуповины |
| Индукция | 40+3, всё в норме; 39 нед «для удобства» | 41+4 с маловодием |
| Экстренные | — | Преэклампсия, эклампсия, отслойка плаценты, предлежание с кровотечением |
| Другое | «Крупный плод» 3800 г, возраст 36 лет, рутинная эпизиотомия, непрерывная КТГ | Задержка роста + допплер, ГСД + макросомия, неудачный вакуум |

### 11 сценариев в справочнике

Справочник (`akusher_prod_v5.html`) содержит 11 базовых сценариев с интерактивными маркерами-чеклистами:

| # | Сценарий |
|---|----------|
| 1 | Окситоцин / стимуляция родовой деятельности |
| 2 | Кесарево из-за снижения активности плода |
| 3 | Зелёные воды — кесарево или срочное вмешательство |
| 4 | Амниотомия — вскрытие плодного пузыря |
| 5 | Срочно вмешаться из-за безводного периода |
| 6 | Кесарево из-за «нет прогресса» / затяжных родов |
| 7 | Фаза замедления — врач предлагает окситоцин |
| 8 | Переходная фаза — врач торопит к потугам |
| 9 | Эпидуральная анестезия / обезболивание |
| 10 | Обвитие пуповиной — предлагают кесарево |
| 11 | Индукция (стимуляция) по сроку / ПДР |

Каждый сценарий содержит две ветки принятия решений с:
- **Маркерами-чеклистом** — отмечаете наблюдаемые признаки
- **Рекомендацией** — появляется после отметки маркеров (ждать / наблюдать / эскалировать)
- **Вопросами врачу** по методу ПРАН
- **Информационными заметками** и развенчанием мифов

### Стоп-сигналы
Красные флаги, при которых нужно немедленно звать врача:
- Сильное вагинальное кровотечение
- Температура выше 38°C
- Выпадение петель пуповины
- Патологическое сердцебиение плода
- Судороги, потеря сознания
- Тяжёлая гипертензия
- Подозрение на отслойку плаценты

### Инструменты
- **Таймер безводного периода** — с момента отхождения вод (меняет цвет при 18ч и 24ч)
- **Счётчик схваток** — подсчёт за 10 минут, средний интервал, определение тахисистолии, сигнал по правилу 5-1-1

### Рутинные процедуры
Справка по стандартным процедурам при поступлении: катетер, влагалищный осмотр, КТГ, окситоцин после родов.

### Словарик терминов
Определения медицинских терминов простым языком: КТГ, безводный период, дистресс плода, фаза замедления, амниотомия, меконий, тахисистолия и др.

## Технологии

- Чистый HTML/CSS/JS — без зависимостей и сборки
- Mobile-first дизайн (оптимизирован под телефон)
- Работает офлайн — один файл, открывается локально
- Нет бэкенда, данные хранятся только в памяти браузера
- Android-обёртка: Kotlin + WebView, Gradle, minSdk 24 (Android 7.0+)

## Использование

### Веб-версия
- **Тренажёр**: открыть `trainer.html` в браузере
- **Справочник**: открыть `akusher_prod_v5.html` в браузере телефона

### Android-приложение

**Скачать готовый APK:** перейти в [Actions](../../actions) → последний успешный билд → скачать артефакт `second-opinion-debug`.

**Собрать самостоятельно:**

```bash
# Требуется: JDK 17, Android SDK
cd android
./gradlew assembleDebug
```

APK будет в `android/app/build/outputs/apk/debug/app-debug.apk`.

**Или открыть в Android Studio:** File → Open → выбрать папку `android/`.

## Дисклеймер

Приложение не является медицинским устройством и не заменяет консультацию врача. Это инструмент для структурированного диалога с медперсоналом и принятия информированных решений в родах.
