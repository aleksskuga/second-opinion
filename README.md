# Second Opinion — Второе мнение в родах

Мобильное веб-приложение для рожениц и их партнёров. Помогает критически оценить предложения врача во время родов и принять осознанное решение: соглашаться на вмешательство или можно подождать.

## Что это

Интерактивный справочник-помощник, который работает прямо в браузере телефона без установки и интернета. Не заменяет врача — даёт структуру для диалога с ним.

## Возможности

### Метод ПРАН
Фреймворк из 4 вопросов врачу перед любым вмешательством (окситоцин, амниотомия, эпидуральная, кесарево):
- **П**ольза — зачем это именно мне сейчас?
- **Р**иски — какие риски у вмешательства?
- **А**льтернатива — есть ли другой способ?
- **Н**ичего — что будет, если подождать?

### 11 клинических сценариев
Каждый сценарий содержит две ветки принятия решений:

| # | Сценарий |
|---|----------|
| 1 | Окситоцин / стимуляция родовой деятельности |
| 2 | Кесарево из-за снижения активности плода |
| 3 | Зелёные воды — кесарево или срочное вмешательство |
| 4 | Амниотомия — вскрытие плодного пузыря |
| 5 | Срочно вмешаться из-за безводного периода |
| 6 | Кесарево из-за «нет прогресса» / затяжных родов |
| 7 | Фаза замедления — врач предлагает окситоцин |
| 8 | Переходная фаза — врач торопит к потугам |
| 9 | Эпидуральная анестезия / обезболивание |
| 10 | Обвитие пуповиной — предлагают кесарево |
| 11 | Индукция (стимуляция) по сроку / ПДР |

Каждая ветка содержит:
- **Маркеры-чеклист** — отмечаете наблюдаемые признаки
- **Рекомендацию** — появляется после отметки маркеров (ждать / наблюдать / эскалировать)
- **Вопросы врачу** по методу ПРАН
- **Информационные заметки** и развенчание мифов

### Стоп-сигналы
Красные флаги, при которых нужно немедленно звать врача:
- Сильное вагинальное кровотечение
- Температура выше 38°C
- Выпадение петель пуповины
- Патологическое сердцебиение плода
- Судороги, потеря сознания
- Тяжёлая гипертензия
- Подозрение на отслойку плаценты

### Инструменты
- **Таймер безводного периода** — с момента отхождения вод (меняет цвет при 18ч и 24ч)
- **Счётчик схваток** — подсчёт за 10 минут, средний интервал, определение тахисистолии, сигнал по правилу 5-1-1

### Рутинные процедуры
Справка по стандартным процедурам при поступлении: катетер, влагалищный осмотр, КТГ, окситоцин после родов.

### Словарик терминов
Определения медицинских терминов простым языком: КТГ, безводный период, дистресс плода, фаза замедления, амниотомия, меконий, тахисистолия и др.

## Технологии

- Чистый HTML/CSS/JS — без зависимостей и сборки
- Mobile-first дизайн (оптимизирован под телефон)
- Работает офлайн — один файл, открывается локально
- Нет бэкенда, данные хранятся только в памяти браузера

## Использование

Открыть `akusher_prod_v5.html` в браузере телефона. Можно сохранить на главный экран как PWA-подобное приложение.

## Дисклеймер

Приложение не является медицинским устройством и не заменяет консультацию врача. Это инструмент для структурированного диалога с медперсоналом и принятия информированных решений в родах.
